# Service-Oriented Architecture Lab

**Описание:**  
Комплексная лабораторная работа по сервисно-ориентированной архитектуре, демонстрирующая эволюцию от простого REST API до полноценной микросервисной архитектуры с использованием различных паттернов и технологий.

**Бизнес-ценность / Применение:**  
Демонстрация практических навыков проектирования и реализации SOA-систем с применением современных архитектурных паттернов. Проект показывает переход от монолитной архитектуры к микросервисам с использованием Spring Cloud, Eureka, Config Server и других enterprise-решений.

**Ключевые функции:**

- [ ] REST API с XML-сериализацией и валидацией данных (Lab 1)
- [ ] React-фронтенд с интеграцией REST API и XML-парсингом (Lab 2)
- [ ] Микросервисная архитектура с Eureka Service Discovery (Lab 3)
- [ ] Spring Cloud Config Server для централизованной конфигурации
- [ ] Load Balancing с Netflix Ribbon
- [ ] SOAP Web Services с Apache CXF (Lab 4)
- [ ] Docker-контейнеризация с docker-compose
- [ ] PostgreSQL с JPA/Hibernate persistence
- [ ] Swagger/OpenAPI документация

**Технологии:**

- **Backend:** Kotlin, Java, Spring Boot, Spring Cloud, JPA/Hibernate
- **Frontend:** React, TypeScript, Gravity UI, Axios
- **Архитектура:** Eureka, Config Server, Netflix Ribbon, Apache CXF
- **База данных:** PostgreSQL
- **Контейнеризация:** Docker, Docker Compose
- **API:** REST, SOAP, XML/JSON сериализация
- **Документация:** Swagger/OpenAPI

**Демо-сценарии:**

- Многоуровневая архитектура с разделением на EJB и Web модули
- Service Discovery и Load Balancing между микросервисами
- Централизованное управление конфигурацией через Config Server
- Интеграция SOAP и REST сервисов в единой экосистеме
